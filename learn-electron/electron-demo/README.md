## electron-demo

学习`electron`构建跨平台的桌面应用程序。

## 快速启动

```
# 进入仓库
cd electron-demo

# 安装依赖库
npm install

# 运行开发环境
npm start

# 运行开发环境 + 加载文件启动
npm start-file

# 运行生产环境
npm start-prod
```
